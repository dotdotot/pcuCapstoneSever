# pcuCapstoneSever

1. API server를 구축하기 위해 FastAPI를 사용하여 서버를 구축
2. FastAPI를 HTTPS로 띄우기 
3. 사용자와 온도, 습도, 미세먼지 등의 방에 대한 정보를 저장하기 위한 데이터베이스 설계 및 구축
